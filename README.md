# Crypto Signals Bot 4H

1. ارفع الملفات إلى GitHub أو Railway.
2. انسخ متغيرات `.env.example` إلى Variables في Railway.
3. أضف `TELEGRAM_BOT_TOKEN` و`TELEGRAM_CHANNEL_ID` و`COINGECKO_API_KEY`.
4. شغّل الخدمة كـ Worker باستخدام `python bot.py`.

تم حذف نظام الثقة وAI Ranking والمتغيرات المكررة الخاصة بمقارنة Stoch RSI وMACD.
